# Stage Guide

Use this guide to choose the right prompt.

| Situation | Stage | Prompt |
|---|---|---|
| You only know the symptom | Diagnose | `01-diagnose.md` |
| You know the bug cause and need fix options | Route | `02-route-known-problem.md` |
| You have an idea but want it challenged | Route | `03-route-with-user-idea.md` |
| You have no idea yet | Route | `04-route-without-user-idea.md` |
| You need a small patch plan | Plan | `05-small-plan.md` |
| You need a multi-file plan | Plan | `06-large-plan.md` |
| You approved a plan | Execute | `07-small-execute.md` or `08-strict-execute.md` |
| Another AI reviewed the plan | Review | `12-review-response-triage.md` |
